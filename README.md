# m2RUg Multilingual Project
This project aims to smoothly use different languages using m2RUg French voicebanks in OpenUtau using tools like dictionaries and additionnal reclists for necessary phonemes.

# Credit
Utau French Ressources for making the m2RUg reclist.
[intunist](https://github.com/intunist/) for the [base dictionnary](https://github.com/intunist/diffsinger-english-support/blob/main/intunist-eng-word-dict.txt) used for english.
